**Number Guessing Game**

A simple Python console game where the player tries to guess a randomly chosen number between 1 and 100.
The player can select a difficulty level that determines how many chances they get to guess the number.

**Features**
- Randomly generates a number between 1â€“100
- Three difficulty levels:
  - Easy (10 chances)
  - Medium (6 chances)
  - Hard (4 chances)
- Provides hints if the guessed number is too high or too low
- Handles invalid (non-numeric) input
- Displays winning and losing messages

**How To Run**
1. Make sure python is installed
2. Clone or download this repo
3. Run the script:
   python main.py
4. Follow the instructions to play
